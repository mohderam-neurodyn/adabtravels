How to deploy on Vercel (quick):
1. Create a GitHub repository named `adabtravels`.
2. Add these files to the repo (upload or git push).
3. Sign in to https://vercel.com with GitHub and "Import Project".
4. Select the `adabtravels` repo and deploy — Framework Preset: "Other" (Static HTML).
5. After deployment you'll have: https://<project-name>.vercel.app
6. Optional: In Vercel dashboard → Domains, set `adabtravels.vercel.app` as project domain name or add custom domain later.

Optional edits:
- Replace the inline SVG logo in index.html with your official logo.svg (put in the repo root and change <svg> to <img src="logo.svg" alt="logo">).
- Replace Formspree action URL in the contact form with your own endpoint or use Vercel Serverless Function.
- Update email/phone/address in the contact section.
